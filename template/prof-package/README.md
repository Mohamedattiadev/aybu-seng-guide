# AYBU SENG Staj Report — Professor Package

A compact bundle to share with the SENG internship commission so they can
see both the blank LaTeX template and a fully filled-in example produced
from it.

## Files

| File | Description |
|------|-------------|
| `aybu-staj-report-template.zip` | The blank LaTeX template (source). Unzip → edit `main.tex` → `pdflatex` twice. |
| `aybu-staj-report-template-preview.pdf` | Compiled PDF of the blank template (placeholders visible). |
| `aybu-staj-report-example-FILLED.pdf` | Compiled PDF of a realistic fictional example (12 pages): cover, TOC, 5 weekly tables with daily entries, General Report (Introduction, 3 projects, Conclusion, Appendices). All names/companies are invented. |
| `aybu-staj-report-example-FILLED-source.zip` | LaTeX source of the example PDF, so the reader can inspect how each field was filled. |
